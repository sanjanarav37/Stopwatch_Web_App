# ⏱️ Stopwatch App

A modern and responsive stopwatch built using **HTML, CSS, and JavaScript**.

Designed with a clean dark interface, smooth animations, and responsive layout to deliver a simple yet elegant user experience.

---

## ✨ Features

- ▶️ Start & Pause Timer
- 🔄 Reset Stopwatch
- 🏁 Record Unlimited Laps
- 🗑️ Clear All Laps
- ✨ Animated Glowing Ring
- 📱 Fully Responsive Design
- 🎨 Smooth Hover Animations
- ⚡ Lightweight & Fast

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
